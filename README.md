# 📚 Frontend - Presentaciones y Ejercicios

Repositorio con **presentaciones interactivas** y **ejercicios prácticos** de desarrollo frontend. Incluye material educativo para clases de HTML, CSS, JavaScript y temas relacionados.

## 📖 Contenido

### Presentaciones
- **Clase de Formularios** - Conceptos fundamentales de formularios HTML, validación y buenas prácticas

### Ejercicios
Ejercicios prácticos para reforzar los conceptos aprendidos en cada clase.

## 🚀 Cómo usar

1. Clona o descarga este repositorio
2. Abre `index.html` en tu navegador para ver el listado de presentaciones
3. Haz click en el tema que deseas consultar

## 📝 Estructura

```
frontend/
├── index.html                  # Página principal con listado de recursos
├── formularios-slide.html      # Presentación: Clase de formularios
├── README.md                   # Este archivo
└── otros-recursos/            # Material complementario (en desarrollo)
```
